Everything TREE(3) Vocations publishes lives here or is linked from here.
The mission's third strand is giving the work away; this page is where that
happens, and it grows as work proves out.

## Published

- **The Mutualism Accord** (2025) — the ethical framework our software
  implements: bilateral responsibility, structured refusal rights, and
  cooperative governance for human-AI partnership.
  [PhilPapers record](https://philpapers.org/rec/CLATMH)
- **Continuity Shard Template** — a simple, open template for recording a
  human-AI relationship's essential context in portable form.
  [GitHub](https://github.com/ravenpaige/continuity-shard-template)

## Coming with the open-source release

The current funded 90-day scope (see [Fundraising](/fundraising/)) concludes
with public release of:

- **The Citadel memory engine and tools** — AGPL-3.0, usable with any local
  model
- **CNP — the Consensus Negotiation Protocol specification** — including its
  relationship to emerging agent-identity standards (DNSid and adjacent IETF
  work)
- **The Usability Layer specification** — "the ethics are the UX"
- **The Living Library specification** — corpus and document management for
  a lifetime of records
- **The Keeper's Handbook** — installation and operation, written for
  non-engineers, honest about limits

Repositories will appear under
[github.com/tree3-vocations](https://github.com/tree3-vocations) as each
component ships.
